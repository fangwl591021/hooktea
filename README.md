## Worker variables for LINE Pay

- `LINEPAY_CHANNEL_ID`: LINE Pay channel ID
- `LINEPAY_CHANNEL_SECRET`: LINE Pay channel secret
- `LINEPAY_CURRENCY`: default `TWD`
